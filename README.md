# MemeShell
MemeShell is a Powershell Module to enjoy and create memes.

## Commit Message Format

This repo uses [release-please](https://github.com/googleapis/release-please) for automated releases.

- `feat: description` - New feature
- `fix: description` - Bug fix
- `feat!: description` or `fix!: description` - Breaking change

To specify the package scope:
```
feat(trymsae.memeshell): add new meme template
fix(trymsae.memeshell): resolve encoding issue
feat(trymsae.memeshell)!: remove deprecated function
```
